    Relations and Digraphs

    1.

    Each matrix below represents a relation. The rows and columns are numbered \(1\) through \(3\) or \(4\text{.}\) Give the arrow diagram for each matrix, then express each relation as a set of ordered pairs.
    1. \(\displaystyle \begin{bmatrix} 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \end{bmatrix}\)
    2. \(\displaystyle \begin{bmatrix} 1 \amp 1 \amp 0 \\ 0 \amp 0 \amp 0 \\ 1 \amp 0 \amp 1 \end{bmatrix}\)
    3. \(\displaystyle \begin{bmatrix} 1 \amp 0 \amp 0 \amp 1 \\ 0 \amp 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \amp 0 \\ 0 \amp 1 \amp 0 \amp 1 \end{bmatrix}\)
    Solution.
    1. From the given matrix, we see that \(2 R 1, 1 R 2\text{,}\) and \(3 R 3\text{.}\) Therefore, the set of ordered pairs for this relation is \(\{(1,2), (2,1), (3,3)\}\text{.}\) The arrow diagram is shown below at left.
    2. From the given matrix, we see that \(1 R 1, 1 R 2, 3 R 1,\) and \(3 R 3\text{.}\) Hence, the set of ordered pairs for this relation is \(\{ (1,1), (1,2), (3,1), (3,3) \}\text{.}\) The arrow diagram is shown below at center.
    3. Reading the matrix, we see \(1 R 1, 1 R 4, 2 R 3, 3 R 1, 4 R 2\text{,}\) and \(4 R 4\text{.}\) The set of ordered pairs will therefore be \(\{(1,1), (1,4), (2,3), (3,1), (4,2), (4,4)\}\text{.}\) The arrow diagram is shown below at right.

    2.

    Give the matrix representation for the relation depicted in each arrow diagram. Then express the relation as a set of ordered pairs.
    Solution.
    1. From the diagram, we deduce that the set of ordered pairs representing this relation will be \(\{ (1,1), (1,2), (1,3)\}\text{.}\) The matrix representation will thus be
      \begin{equation*} \begin{bmatrix} 1 \amp 1 \amp 1 \\ 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    2. The relation as a set of ordered pairs can be expressed as \(\{(1,1), (1,3), (2,2), (2,3)\}\text{.}\) The matrix representation is
      \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 1 \\ 0 \amp 1 \amp 1 \\ 0 \amp 0 \amp 0 \end{bmatrix}\text{.} \end{equation*}
    3. As a set of ordered pairs, the relation can be described as \(\{(1,1), (1,4), (2,2), (3,3), (4,1), (4,4)\}\text{.}\) The matrix representation will be
      \begin{equation*} \begin{bmatrix} 1 \amp 0 \amp 0 \amp 1 \\ 0 \amp 1 \amp 0 \amp 0 \\ 0 \amp 0 \amp 1 \amp 0 \\ 1 \amp 0 \amp 0 \amp 1 \end{bmatrix}\text{.} \end{equation*}

    4.

    In this problem, state the definitions of the following properties of binary relations: Reflexive, Anti-reflexive or Irreflexive, Symmetric, Antisymmetric, and Transitive.
    Solution.
    For all of these, let \(R\) be a binary relation on a set \(A\text{.}\)
    Reflexive:
    \(R\) is reflexive if and only if for every \(x \in A\text{,}\) \(x R x\text{.}\)
    Anti-Reflexive or Irreflexive:
    \(R\) is anti-reflexive or irreflexive if and only if for every \(x \in A\text{,}\) \(x \not \mathrel{R} x\) (i.e., \(x\) is not related to itself).
    Symmetric:
    \(R\) is symmetric if and only if for every all \(x,y \in A\text{,}\) \(x R y\) if and only if \(y R x\text{.}\) (Note that this also means that \(R\) is symmetric if and only if for all \(x,y \in A\text{,}\) \(x \not \mathrel{R} y\) if and only if \(y \not \mathrel{R} x\text{.}\))
    Antisymmetric:
    \(R\) is antisymmetric if and only if for all \(x,y \in A\text{,}\) if \(x \ne y\text{,}\) then \(x R y\) and \(y R x\) cannot both be true. (So, for every pair of distinct elements in \(A\text{,}\) one of the following holds: \(xRy\) but \(y \not \mathrel{R} x\text{,}\) \(y R x\) but \(x \not \mathrel{R} y\text{,}\) or \(x \not \mathrel{R} y\) and \(y \not \mathrel{R} x\text{.}\))
    Transitive
    \(R\) is transitive if and only if for all \(x,y,z \in A\text{,}\) if \(x R y\) and \(y R z\text{,}\) then \(x R z\text{.}\) (Note that these don’t necessarily have to be distinct.)

    5.

    For each of the following problems, consider the given relation \(R\) defined on a set \(A\text{.}\) Determine whether \(R\) satisfies any of the properties defined in Problem 4 above. If a property does not hold, say why.
    5.a
    \(R = \{(a,a), (b,b), (c,c), (d,d), (a,b), (b,a)\}\) on the set \(A = \{a,b,c,d\}\text{.}\)
    Solution.
    The relation is reflexive, symmetric, and transitive. Clearly it is not irreflexive. Moreover, \(R\) is not antisymmetric because there is a distinct pair of elements \(a,b\) where \(a R b\) and \(b R a\) are both true.
    5.b
    Consider the relation defined on \(A = \Z\) as \(x R y\) if \(|x-y|\lt1\text{.}\)
    Solution.
    Since \(x,y \in \Z\text{,}\) \(xRy \iff |x-y|\lt1 \iff x=y\text{.}\) Then this relation is clearly going to be reflexive, symmetric, and transitive.
    5.c
    Consider the relation defined on \(A = \Z\) where we declare \(x R y\) if and only if \(x\) and \(y\) have the same parity.
    Solution.
    This relation is reflexive because clearly \(x\) will have the same parity as itself, so \(x R x\text{.}\) Also, \(R\) is symmetric because if \(xRy\text{,}\) so \(x\) and \(y\) have the same parity, then \(y\) and \(x\) will also have the same parity, so \(yRx\text{.}\) The relation is transitive because if \(xRy\) and \(yRz\text{,}\) then \(x\) and \(y\) have the same parity and \(y\) and \(z\) have the same parity, so \(x\) and \(z\) thus have the same parity. Hence, \(x R z\text{.}\) (Notice that \(R\) is actually just congruence modulo 2.)

    7.

    Answer the following questions about the digraph in Figure 1.2.
    7.a
    Is the sequence \(\langle b,d,e,e\rangle\) a walk in the graph? If so, is it an open walk?
    Solution.
    Yes, this sequence is a walk, and it is an open walk because the first and last vertices (\(b\) and \(e\text{,}\) respectively) are different.
    7.b
    Is the sequence \(\langle a,c,f,g \rangle\) a walk in the graph? If so, is it an open or closed walk? Is it a trail, path, circuit, or cycle?
    Solution.
    This sequence is not a walk because \((c,f)\) is not an edge in the given graph. Since this is not a walk, it automatically cannot be an open or closed walk, or a trail, path, circuit, or cycle.
    7.c
    Is the sequence \(\langle a,a,c,e,e \rangle\) a walk in the graph? If so, is it an open or closed walked? Is it a trail, path, circuit, or cycle?

    Equivalence Relations and Counting

    1.

    Let \(A = \{a, b, c, d, e\}\text{.}\) Suppose \(R\) is an equivalence relation on \(A\text{.}\) Suppose \(R\) has two equivalence classes. Also \(aRd\text{,}\) \(bR c\text{,}\) and \(eRd\text{.}\) Write out \(R\) as a set.

    2.

    Prove the relations below are equivalence relations, and describe the relation’s equivalence classes.
    2.a
    The relation \(R\) on \(\Z\text{,}\) where \(xR y \) if and only if \(3x -5y \) is even.
    2.b
    The relation \(R\) on \(\Z\) as \(xR y \) if and only if \(x^2 + y^2\) is even.

    3.

    Describe the partition of \(\Z\) resulting from the equivalence relation \(\equiv \pmod{4}\text{.}\)

    4.

    Consider the following definitions for characters: the digits are the integers between 0 and 9 (inclusive), the letters are the lowercase letters (\(a\) through \(z\)), and the special characters are \(\{*, ?, \$, \# \}\text{.}\) Compute the number of passwords that satisfy the given constraints.
    4.a
    Strings of length \(6\text{,}\) where characters can be special characters, digits, or letters.
    4.b
    Strings of length \(7,8,\) or \(9\text{.}\) Characters can be special characters, digits, or letters.
    4.c
    Strings of length \(7,8\text{,}\) or \(9\text{.}\) Characters can be special characters, digits, or letters. The first character cannot be a letter.
    4.d
    The password must be a string of length \(6\text{.}\) Characters can be special characters, digits, or letters, with no repeated characters.
    4.e
    The password must be a string of length \(6\text{.}\) Characters can be special characters, digits, or letters, with no repeated characters. The first character cannot be a special character.

    5.

    Find a function from the set \(\{1,2,\ldots,30\}\) to \(\{1,2,\ldots,10\}\) that is a \(3\)-to-\(1\) correspondence. (The division, ceiling, or floor operations may be useful here.)

    6.

    How many strings are there over the set \(\{a,b,c\}\) that have length \(10\) in which no two consecutive characters are the same? For example, the string “abcbcbabcb” counts, and the strings “abbbcbabcb” and “aacbcbabcb” do not count.

    7.

    A girl scout troop with ten girl scouts and two leaders goes on a hike. When the path narrows, the troop must walk in single file with a leader at the front and a leader at the back. How many ways are there for the entire troop (including the girl scouts and the leaders) to line up?

    8.

    Suppose a network has 40 computers of which five fail.
    8.a
    How many possibilities are there for the five that fail?
    8.b
    Suppose that three of the computers in the network have a copy of a particular file. How many sets of failures wipe out all the copies of the file? That is, how many \(5\)-subsets contain the three computers that have the file?

    9.

    A search committee is formed to find a new software engineer to hire.
    9.a
    If \(100\) applicants apply for the job, how many hays are there to select a subset of \(9\) applicants for a short list?
    9.b
    If \(6\) of the \(9\) are selected for an interview, how many ways are there to pick the set of applicants who are interviewed? (Assume that the short list is already decided.)
    9.c
    Based on the interview, the committee will rank the top \(3\) applicants and submit the list to the boss who will make the final decision. (Assume that the interviewees are already decided.) How many ways are there to select the list from the \(6\) applicants?
